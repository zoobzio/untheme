# Examples

Runnable demonstrations of untheme in a real setting. Each example consumes the public [`untheme`](../packages/untheme) package and an integration from [`../integrations`](../integrations).

| Example                      | Directory            | Description                                              |
| ---------------------------- | -------------------- | ------------------------------------------------------- |
| [nuxt](./nuxt)               | `examples/nuxt`      | Nuxt landing page restyled live across aurora themes    |
| [terrazzo](./terrazzo)       | `examples/terrazzo`  | DTCG token JSON compiled to an `untheme.config.ts`      |
